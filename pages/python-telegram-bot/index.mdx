## **PTB** tuzilishi

**PTB** `telegram` va `telegram.ext` modullaridan iborat.

### `telegram` moduli

Ushbu modulda faqat [Telegram Bot API](https://core.telegram.org/bots/api) bilan ishlash uchun "o‘ralma" mavjud.
O‘ralma deganda Python tilida siz uchun qulay `class`’lar, `type`’lar va hokazo yozilgan kodlarni nazarda tutyapman.

Misol uchun:

```python
import asyncio
from telegram import Bot

async def main():
    bot = Bot(token='im123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11') # Shu yerga o‘z token’ingizni qo‘ying
    await bot.sendMessage(chat_id=1234567, text="OnaBot.uz dan salom!") # O‘zingizni chat_id’ingizni qo‘ying

if __name__ == "__main__":
    asyncio.run(main())
```
